## Conventions de nommage

| Format | Exemple |
| :--- | :--- |
| Nom | `kebab-case` minuscules |
| Extension | `.png` (préféré) ou `.jpg` |
| Taille | < 1 Mo si possible |
| Résolution | 1920x1080 ou crop ciblé |

## Outils recommandés

- **Windows** : Win + Shift + S (Snipping Tool)
- **VS Code** : Extension "Polacode" pour code
- **Chrome** : Extension "GoFullPage" pour pages web
- **Annotation** : Snagit ou Markup Hero
